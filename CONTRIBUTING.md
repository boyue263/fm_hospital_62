# 贡献指南

感谢您考虑为 FM Hospital 项目做出贡献！

## 开发流程

1. Fork 仓库
2. 创建功能分支 (\git checkout -b feature/AmazingFeature\)
3. 提交更改 (\git commit -m 'Add some AmazingFeature'\)
4. 推送到分支 (\git push origin feature/AmazingFeature\)
5. 创建 Pull Request

## 分支管理

- \master\ - 稳定版本
- \develop\ - 开发主干
- \eature/*\ - 功能开发
- \elease/*\ - 版本发布

## 代码规范

- 遵循 Java 编码规范
- 提交信息使用约定式提交
- 新功能需要包含单元测试
